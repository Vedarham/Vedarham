## Projects
 
**[InLeagle — Indian Banking Law AI Assistant](https://github.com/Vedarham/InLeagle01)**
RAG-based legal assistant using InLegal-Sbert embeddings and Qdrant Cloud, covering 8 Supreme Court judgments, 6 banking acts (SARFAESI, IBC, NI Act), and RBI guidelines. FastAPI backend, deployed via Docker on HuggingFace Spaces.
`Python · FastAPI · RAG · Qdrant · Groq LLaMA · Docker`
[Demo](https://huggingface.co/spaces/vedastra/InLeagle-banking-law) · [Code](https://github.com/Vedarham/InLeagle01)
 
**[FraudShield — AI Scam Detection](https://github.com/Vedarham/fraudshield01)**
Multilingual fraud/phishing detector (English, Hindi, +) built on a fine-tuned DistilBERT model, trained on 10,000+ labeled messages, reaching 92% accuracy. Shipped as a Chrome extension with a FastAPI backend.
`Python · DistilBERT · FastAPI · Chrome Extension API`
[Demo](https://fraudshield01-iota.vercel.app/) · [Code](https://github.com/Vedarham/fraudshield01) · [Model](https://huggingface.co/vedastra/scam-phish-multilingual-model)
 
**[Kalastra — AI-Enhanced E-Commerce](https://github.com/Vedarham/kalastra0)**
Voice-powered marketplace for artisans: Deepgram speech-to-text for listings, Gemini API for product enrichment, Stripe payments, JWT auth, and semantic search.
`MERN · TypeScript · Gemini AI · Stripe · Deepgram`
[Demo](https://kalastra0-client.vercel.app/) · [Code](https://github.com/Vedarham/kalastra0)
 
**[Xapad — Real-Time Chat with AI Bot](https://github.com/Vedarham/Xapad)**
WebSocket chat application with an integrated RAG-powered assistant (XaBot) backed by a custom knowledge base, and persistent message storage.
`Node.js · Express · Socket.IO · MongoDB · RAG`
[Demo](https://xapad.vercel.app/) · [Code](https://github.com/Vedarham/Xapad)
